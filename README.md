# Zapret

**Internet censorship bypass tool for Windows**

Zapret is a network filtering tool designed to help users access websites and services that may be restricted by ISP-level filtering and Deep Packet Inspection (DPI) systems.

## Features

* ✅ Access to Discord (voice and text channels)
* ✅ Access to YouTube without ISP restrictions
* ✅ Access to Telegram messaging services
* ✅ DPI bypass support for various other online services
* ✅ Lightweight and easy to use

## System Requirements

* Windows 10 / 11
* No additional software required
* Minimal system resource usage

## Installation

1. Download the latest version from the **Releases** section.
2. Run `Zapret.exe`.
3. Wait for the automatic configuration process to complete.
4. Enjoy unrestricted access to supported services.

## How It Works

Zapret utilizes traffic filtering and packet manipulation techniques to reduce the effectiveness of DPI-based blocking methods used by some internet service providers.

The application automatically applies the required network configuration and operates in the background with minimal user interaction.
